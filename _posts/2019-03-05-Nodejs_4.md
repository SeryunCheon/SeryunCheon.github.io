# 스트림
 - 스트림: 데이터의 전송 흐름임(데이터 입력 부분과 출력 부분사이의 흐름~~~)
   - 콘솔 입출력, 파일 읽기쓰기, 서버/클라이언트끼리의 데이터전송에 쓰임
   - 스트림모듈은 이러한 위의 기능들을 가능하게 하는 추상인터페이스임 이걸 이용해서 잘 쓰면 됨.
 - 스트림종류
   - 읽기 스트림 : Readable Stream
   - 쓰기 스트림 : Writable Stream
   - 읽기/쓰기 : Duplex
   - 변환 : Transform
